### Week 4 Objectives:
* Understand AWS Virtual Private Cloud (VPC) and its key components: Subnet, Route Table, and Gateway.  
* Practice creating a custom VPC and deploying EC2 instances in both public and private subnets.  
* Learn about network security mechanisms such as Security Groups and Network ACLs.
